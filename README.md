# FaceCheck-Project
FaceAttend – Face Recognition Attendance System

FaceAttend is an AI-powered smart attendance system that leverages YOLOv8 for real-time face detection and a deep learning face recognition model for student identification. The system automatically marks attendance by scanning live camera feeds, reducing manual errors and preventing fraudulent check-ins.

Key Features:

Real-time face detection using YOLOv8.

Face recognition with deep learning (FaceNet/DeepFace/InsightFace).

Automatic attendance logging into the database.

Web dashboard for teachers to monitor and generate reports.

Role-based access (Admin, Teacher, Student).

Scalable and deployable via Docker.

This project aims to enhance efficiency and accuracy in classrooms, replacing traditional manual or RFID-based attendance systems.
